# Agenda de Contatos em C ANSI

Sistema simples de gerenciamento de contatos desenvolvido em C ANSI para a disciplina de Fundamentos de Programação.

## Funcionalidades

- Inclusão de contatos
- Listagem de contatos
- Consulta de contatos pelo nome
- Exclusão de contatos

## Estrutura do projeto

agenda-contatos-c/
├── README.md
├── src/
├── docs/
└── exemplos/

## Como compilar

```bash
gcc src/agenda_contatos.c -o agenda
```

## Como executar

```bash
./agenda
```

No Windows:

```bash
agenda.exe
```
